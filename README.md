# PMShri Dharmanagar Govt. Girls' H.S. School Website  
A modern, responsive website built with React for the PMShri Dharmanagar Govt. Girls’ Higher Secondary School — showcasing academics, events, facilities, gallery and school information.

## About  
This website is designed for the PMShri Dharmanagar Govt. Girls’ Higher Secondary School. It aims to provide a clean, professional UI and a mobile-friendly experience to visitors — students, parents, staff, and the wider community. It includes sections like:  
- Academics (courses, departments)  
- Facilities (labs, library, campus)  
- Events & Gallery (school functions, photos)  
- Contact & About us  

## Features  
- Fully responsive design — works well across devices  
- Easy navigation between sections/pages  
- Optimised performance and lightweight build  
- Structured code organisation with React components  
- Deployable to common static-hosting platforms  

## Tech Stack  
- [React.js](https://reactjs.org) (frontend UI)  
- JavaScript (ES6+)  
- HTML5 & CSS3 (styling, layout)  
- Build tools & dependencies managed via `npm`
