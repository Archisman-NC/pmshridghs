PMShri Dharmanagar Govt. Girls' H.S. School Website

A modern and responsive React.js website for PMShri Dharmanagar Govt. Girls' Higher Secondary School, showcasing academics, events, facilities, and school information.

Features

Clean, professional UI

Fully responsive design

Sections for academics, facilities, events, and gallery

Smooth navigation & optimized performance
